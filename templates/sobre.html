Sobre o Dashboard-TRONIK

Sistema de monitoramento inteligente de coletores de resíduos eletrônicos

Sobre o Projeto

O Dashboard-TRONIK é um sistema de monitoramento de coletores inteligentes que permite visualizar em tempo real o nível de preenchimento e status de cada lixeira equipada com sensores ultrassônicos.

Desenvolvido como uma solução web completa para monitoramento de coletores, o projeto permite que equipes aprendam desenvolvimento web moderno através de um projeto real e prático, utilizando tecnologias como Python Flask, HTML5, CSS3 e JavaScript.

Sobre a TRONIK

A TRONIK é uma empresa especializada na coleta e reciclagem de resíduos eletrônicos, atuando tanto em residências quanto em empresas. Nosso processo inclui:

  • Coleta: Recolhemos resíduos eletrônicos em casas e empresas
  • Parcerias: Enviamos os materiais coletados para parceiros especializados
  • Separação: Realizamos a separação de vidro, plásticos, componentes, cobre e alumínio
  • Reaproveitamento: Enviamos os materiais separados aos fabricantes de eletrônicos para reaproveitamento

Objetivo

Desenvolver uma solução web completa para monitoramento de coletores, permitindo que a equipe aprenda desenvolvimento web moderno através de um projeto real e prático. O sistema visa:

  • Visualização em tempo real do nível de preenchimento das lixeiras
  • Alertas automáticos quando o nível ultrapassa 80%
  • Relatórios detalhados de coletas e níveis por período
  • Interface web responsiva para desktop e mobile
  • Otimização de rotas de coleta baseada em dados

Modelo de Negócio

B2B2C

A TRONIK opera com modelo de negócio B2B2C (Business-to-Business-to-Consumer), conectando empresas, instituições e consumidores finais através de uma plataforma de coleta e reciclagem de resíduos eletrônicos.
